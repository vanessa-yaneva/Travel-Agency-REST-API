## :running: Run Locally

Clone the project

```bash
  git clone https://github.com/vanessa-yaneva/Travel-Agency-REST-API.git
```

Go to the project directory

```bash
  cd Travel-Agency-REST-API
```

Install packages

```bash
  npm install
```

Start the server

```bash
  npm start
```

:pushpin: The server will run on http://localhost:8080/
